Chatsome

Chatsome is a chat program which allows you to communicate over LAN or even the Internet. With a simple userfriendly interface and lots of functions Chatsome can be used by everyone and everywhere.

Features

  • Easy to use
  • Many Twitch.tv emotes are supported
  • Commands (mute, kick, ban ...)
  • Custom messages (join, leave ...)
  • Simple to configurate
  • No installation required
  • It's all free!

How to

In order to execute any of the Chatsome programs follow these steps:

  • Download the ConnectionProvider.dll
  • Place this file in the same folder as your Chatsome programs are going to be placed

To get started follow these few steps:

  • Download the Chatsome.exe
  • Execute the Chatsome.exe
  • Enter the IP-Adress or Computername of the host
  • Press 'Connect'
  • Chat with your friends!

In case you want to host a Chatsome-Server follow these steps:

  • Download the Chatsome_server_host.exe
  • Execute the Chatsome_server_host.exe
  • Press 'Host'
  • Now your friends can connect!
